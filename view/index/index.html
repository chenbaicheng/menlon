Medoo-MVC使用

  • config

    • 数据库配置文件位置在"/config/db.config.php"，正确填写配置后可进行其他操作。

  • Controller

    • 通过"/index.php?c=index&a=hello"可访问indexController的hello方法。

  • View

    • 在yourController中的hello方法通过"$this->display( $datas )"将$datas数据传入模板，模板文件:"/view/your/hello.html。"

  • Model

    • 在yourController中通过"new your()"来实例化your，model文件位置："/model/your.model.php"。

URL Rewrite

  • 设置后效果

    • 1.通过“http://yourSiteUrl/index/hi”访问indexController->hi()
    2.访问“http://yourSiteUrl/index/hi/param1/value1/param2/value2”可将参数“param1”和“param2”传递到indexController的function hi()

  • 配置

    • 1.修改配置文件“/config/app.config.php”中的“APP_PATH”和“URL_PATHINFO”配置项
    2.添加服务器Rewrite规则

    Apache：
    RewriteEngine on
    RewriteCond %{REQUEST_FILENAME} !-d
    RewriteCond %{REQUEST_FILENAME} !-f
    RewriteRule ^(.*)$ index.php/$1 [QSA,PT,L]

    Nginx：
    location / {
        root /rootPath/;
        index index.php index.html index.htm;
        if (!-e $request_filename) {
        rewrite ^/(.*)$ /index.php?s=$1 last;
        }
    }
